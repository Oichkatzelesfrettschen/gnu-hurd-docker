# Source Materials Directory

This directory contains source materials, upstream files, and reference data.

## Purpose

Store source files that are:
- Downloaded from upstream (Debian images, etc.)
- Reference materials for development
- Unmodified third-party sources
- Research data and artifacts

## Organization

- Keep upstream sources clearly labeled
- Document source URLs and versions
- Use subdirectories for different sources
- See `scripts/setup-hurd-amd64.sh` for image download process

## Related

- Images are stored in `images/` directory
- Compiled artifacts go in `build/` directory
- Documentation goes in `docs/` directory
